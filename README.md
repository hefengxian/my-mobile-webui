# KMW Mobile Web Client

KWM 移动网页版，弥补 APP 可能在平台受限的问题


## 功能规划

- [ ] 登录
- [ ] 分类信息浏览；查看、筛选、操作分类信息
- [ ] 搜索（Solr）；全库快速搜索
- [ ] 分析/报告；查看操作结果，分析报告
- [ ] 个人/设置；个人设置、账号管理、系统设置等
- [ ] 文章详情页面；操作信息
- [ ] 可以通过手机的添加到桌面可以类似原生 APP（要研究）
